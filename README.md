
# 🏏 BPL-DREAM 11  
**Assemble Your Ultimate Dream 11 Cricket Team**

An interactive, high-performance web application designed for cricket enthusiasts to build and manage their fantasy teams. Built with the **MERN** stack philosophy (React, Tailwind CSS, and daisyUI), this project mimics professional fantasy league platforms with real-time coin management and dynamic player selection.

## ✨ Key Features

- **Dynamic Player Selection**  
  Browse a curated list of top-tier cricket players with detailed stats (batting/bowling styles and price).

- **Coin-Based Economy**  
  Users start with a virtual balance. Players can be "purchased" for the team, and credits are automatically deducted.

- **Credit Claim System**  
  A beautiful hero banner allowing users to claim free credits to boost their balance.

- **Smart Validation**
  - Prevents selecting the same player twice  
  - Enforces a maximum squad limit of 11 players  
  - Ensures sufficient balance before purchase  

- **Newsletter Integration**  
  Floating newsletter component with backdrop blur effect.

- **Responsive UI**  
  Fully optimized for mobile, tablet, and desktop using **Tailwind CSS** and **daisyUI**.

- **Real-Time Notifications**  
  Integrated **React-Toastify** for instant feedback.

---

## 🛠️ Tech Stack

- **Frontend:** React.js (Vite)  
- **Styling:** Tailwind CSS & daisyUI  
- **Icons:** React Icons  
- **State Management:** React Hooks (`useState`, `useEffect`)  
- **Notifications:** React-Toastify

## 📦 Installation & Setup

### 1. Clone the repository
```bash
git clone https://github.com/nmjakaria/BPL-Dream11-web-first.git
````

### 2. Navigate to project directory

```bash
cd BPL-Dream11-web-first
```

### 3. Install dependencies

```bash
npm install
```

### 4. Run the development server

```bash
npm run dev
```

### 5. Open in browser

```
http://localhost:5173
```
